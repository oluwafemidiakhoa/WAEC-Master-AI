## ğŸ® How to Use

### Practice Mode
1. Click **"Smart Practice"** on the home screen
2. Choose your subject (Math, English, or Biology)
3. Select topics or choose "All Topics" for adaptive practice
4. Answer questions and get instant feedback
5. Review explanations and track your progress

### Mock Exam Mode
1. Click **"Mock Exam"** from the home screen
2. Configure exam settings:
   - Number of questions (20, 40, or 60)
   - Subject distribution
   - Difficulty preference
3. Take the timed exam under realistic conditions
4. Review comprehensive results and performance analytics

### Online Access

1. Clone this repository:

   ```bash
   # Using Python

   ```

1. Visit the app in Chrome, Edge, or Safari
2. Click the install prompt or "Add to Home Screen"
3. Use like a native app with offline capabilities
